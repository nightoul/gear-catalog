# STRYMON BIGSKY

Digital reverb pedal.

It has 12 reverb algorithms (or machines):

- Room
- Hall
- Plate
- Spring
- Swell
- Bloom
- Cloud
- Chorale
- Shimmer
- Magneto
- Nonlinear
- Reflections

Only one algorithm can be selected at a time. The BigSky sounds can be broken down into two components: the first is the base algorithm (machine). The second component is the combination of parameters (mostly knobs) which are controlled by the user. Together they make the final sound and can be saved as presets.
