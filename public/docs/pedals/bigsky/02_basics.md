## THE BASICS



### MODES OF OPERATION

Strymon BigSky does not define any explicit modes of operation. The closest similar concept is toggling the display view: when pushing the **Type** knob, the display shows either the active preset name or the Decay value.



### BANKS AND PRESETS

The pedal has 300 preset slots organized into 100 banks (numbered 0-99). Each bank has 3 presets.

The pedal has three footswitches and each one corresponds to a preset in a bank. A green LED indicates that the preset is active; an orange LED indicates that the preset is being tweaked.

#### Factory presets

The unit ships with 100 factory presets stored from 00A to 33A. After slot 33A they repeat. The factory presets are great and can be used without any additional tweaking.

#### Cycling through presets

- by turning the **Value knob**
- by stepping on footswitches: A + B footswitches select lower bank, B + C footswitches select higher bank. When bank is selected, then footswitches A, B or C select a preset of that bank.



### SAVING A PRESET

#### To save to a current location

1. Hold the Type knob
2. Display shows SAVE
3. Push the A or B or C footswitch to save preset on either A or B or C location

#### To save to a new location

1. Hold the Type knob
2. Display shows SAVE
3. Turn the Value knob to select a bank
4. Push the A or B or C footswitch to save preset on either A or B or C location

#### To cancel save

To cancel an in-progress save, push the Value knob.
